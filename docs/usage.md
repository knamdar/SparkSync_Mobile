# Usage walkthrough

## Add a device
- Open Settings, then Devices, then Add New.
- Enter host, optional Tailscale host, port, username, and password.

## Connect to a device
- On the home screen, select the device and tap Connect.
- Toggle Tailscale if you want to use its hostname or IP.

## Use the terminal
- In Settings, enable Terminal under Apps.
- Return to the device workspace and open Terminal.

## Add custom actions
- While connected, open the Custom section and select Add New.
- Choose URL or script, then decide whether to link to a URL or override the host and port.
