# FAQ

## Does SparkSync bundle any devices or demo connections?
No. All devices and custom actions are created locally by the user.

## Where are passwords stored?
Passwords are stored in the device Keychain.

## Can I share custom actions across devices?
Custom actions are created per device. Use the recipes in `docs/custom-actions.md` to recreate them consistently.

## Is there a way to avoid showing real hosts in screenshots?
Yes. Use placeholder values and a clean install before recording or capturing screenshots.
