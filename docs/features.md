# Features

## Connection management
- Device list with friendly names and optional Tailscale hosts.
- Per-device port and credential settings.
- Connection timeout controls.

## Terminal access
- Embedded terminal for live sessions.
- Quick launch from the device workspace.

## Custom actions
- URL or script-based actions per device.
- Optional host and port override for each action.
- One tap to open a browser or trigger a scripted task.

## Privacy and safety
- No bundled devices or customs.
- Credentials stored in Keychain.
- User data is cleared on app delete.
